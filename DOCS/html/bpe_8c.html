int inicializarTokensUnicos ( MatrizTexto m_texto,
MatrizTokens * m_tokens )

Inicializa o alfabeto com todos os caracteres únicos presentes no texto.

  • Esta função percorre todo o corpus de texto e identifica os caracteres individuais (bytes) únicos. Cada caractere único é adicionado como um token inicial ao vocabulário. É o primeiro passo do algoritmo BPE.
  • Parameters
    m_textoA matriz contendo o corpus de texto original.
    m_tokensPonteiro para a matriz de tokens onde os caracteres serão guardados.
    Returns
    int Retorna 1 em caso de sucesso, 0 se houver erro de alocação.
  • Note
    Complexidade: O(N), onde N é o número total de caracteres no texto, pois faz uma leitura linear completa do corpus.
