Redimensiona a matriz de texto quando esta fica cheia. Utiliza realloc para aumentar a capacidade do array de ponteiros.

Parameters
mPonteiro para a matriz a redimensionar.
nova_capacidadeO novo tamanho total desejado.
Returns
int 1 em caso de sucesso, 0 se falhar a alocação de memória.
Note
Complexidade: O(N) onde N é o número de linhas, devido à cópia de ponteiros pelo realloc.
