MatrizTokens tokens,

Converte uma string numa sequência de IDs de tokens usando "Greedy Longest-Match". Percorre a frase e, em cada posição, tenta encontrar o maior token possível do vocabulário que coincida com o texto atual.

Parameters
fraseA string de entrada.
tokensO vocabulário disponível.
n_idsPonteiro de saída para armazenar o número de tokens encontrados.
Returns
int* Array dinâmico contendo os IDs dos tokens (deve ser libertado com free).
Note
Complexidade: O(L * V), onde L é o tamanho da frase e V o tamanho do vocabulário, pois testa todos os tokens em cada posição.
