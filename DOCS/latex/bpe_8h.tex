Aplica a fusão de um par de tokens em todo o corpus de texto. 


\begin{DoxyItemize}
\item Percorre todas as linhas do texto e substitui todas as ocorrências da sequência {\ttfamily token1} seguido de {\ttfamily token2} pelo novo token fundido. As linhas de texto são realocadas para acomodar as alterações.
\item

\hline

\hline
{\em token2} & O segundo token do par (ex\+: "{}r"{}). \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
int Retorna o número total de fusões realizadas, ou -\/1 em caso de erro.
\end{DoxyReturn}

\item \begin{DoxyNote}{Note}
{\bfseries{Complexidade\+:}} O(\+L), onde L é o comprimento total do texto, pois envolve. scanning linear e reconstrução das strings. 
\end{DoxyNote}

Inicializa o alfabeto com todos os caracteres únicos presentes no texto. 


\begin{DoxyItemize}
\item Esta função percorre todo o corpus de texto e identifica os caracteres individuais (bytes) únicos. Cada caractere único é adicionado como um token inicial ao vocabulário. É o primeiro passo do algoritmo BPE.
\item 
\begin{DoxyParams}{Parameters}
{\em m\+\_\+texto} & A matriz contendo o corpus de texto original. \\
\hline
{\em m\+\_\+tokens} & Ponteiro para a matriz de tokens onde os caracteres serão guardados. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
int Retorna 1 em caso de sucesso, 0 se houver erro de alocação.
\end{DoxyReturn}

\item \begin{DoxyNote}{Note}
{\bfseries{Complexidade\+:}} O(\+N), onde N é o número total de caracteres no texto, pois faz uma leitura linear completa do corpus. 
\end{DoxyNote}
